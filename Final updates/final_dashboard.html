ls Command

The ls command is used to list directory contents.

cd Command

The cd command is used to change the current directory.

pwd Command

The pwd command prints the current working directory.

mkdir Command

The mkdir command is used to create directories.

rmdir Command

The rmdir command is used to remove empty directories.

cp Command

The cp command is used to copy files and directories.

mv Command

The mv command is used to move or rename files and directories.

rm Command

The rm command is used to remove files or directories.

touch Command

The touch command is used to create empty files or update the timestamp of existing files.

find Command

The find command is used to search for files in a directory hierarchy.

ps Command

The ps command is used to display information about active processes.

top Command

The top command is used to display real-time information about running processes.

df Command

The df command is used to display disk space usage.

du Command

The du command is used to estimate file space usage.

uptime Command

The uptime command is used to tell how long the system has been running.

ping Command

The ping command is used to check the network connectivity between two nodes.

netstat Command

The netstat command is used to display network connections, routing tables, and interface statistics.

ss Command

The ss command is used to dump socket statistics.

curl Command

The curl command is used to transfer data from or to a server using various protocols.

wget Command

The wget command is used to download files from the web.

useradd Command

The useradd command is used to create a new user or update default new user information.

usermod Command

The usermod command is used to modify a user account.

passwd Command

The passwd command is used to change a user's password.

groups Command

The groups command is used to show which groups a user is a member of.

chmod Command

The chmod command is used to change file access permissions.

chown Command

The chown command is used to change file owner and group.

chgrp Command

The chgrp command is used to change group ownership.

umask Command

The umask command is used to set file mode creation mask.

systemctl Command

The systemctl command is used to examine and control the systemd system and service manager.
